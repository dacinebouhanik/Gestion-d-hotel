#  Système de Gestion d'Hôtel (Java POO)

Projet de gestion d'hôtel développé en Java dans le cadre d'un apprentissage de la Programmation Orientée Objet (POO).

##  Description

Cette application permet de gérer :

- Les chambres
- Les clients
- Les réservations
- Les paiements

Le projet est structuré selon le modèle MVC :
- `model` : logique métier et entités
- `vue` : interface utilisateur
- `controller` : gestion des interactions

##  Technologies utilisées

- Java
- Programmation Orientée Objet (POO)
- Architecture MVC

##  Fonctionnalités principales

- Création et gestion des clients
- Réservation de chambres
- Vérification de disponibilité
- Gestion des paiements
- Interface interactive

## 📂 Structure du projet
```
src/
├── controller/
├── model/
└── vue/
```
